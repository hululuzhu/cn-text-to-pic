# 趣味中文生成图片 demo
利用现成的语言和图片生成及鉴别模型来接受中文输入输出图片。

- 中文到英文采用 HuggingFace Helsinki-NLP
- 英文到图片生成采用 VQGAN
- 图片鉴别使用OpenAI的CLIP
- VQGAN和CLIP合起来有一个pixray project
